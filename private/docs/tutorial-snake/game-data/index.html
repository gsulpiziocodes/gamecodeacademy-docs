Game Data

Now that we have a basic structure for our game laid out, we need to think about what kind of data exists in Snake.

The Building Blocks#

To do so, let's list out the objects in the game:

  • The Snake
  • The Food

That's about it! luckily we won't have too much data to deal with.

Now, let's dive a little deeper and think about the attributes each of these items has.

The Snake#

  • Position for each tail piece
  • Length of tail
  • Direction the snake is traveling

The Food#

  • Position

Data to Code#

Defining Structures#

With these attributes identified, we can go ahead and translate this data into code that our game can run off of.

In C there are things called structs, which are basically just packages of data that we can interact with to bundle data together. We'll use structs to represent our Snake and our Food.

Go ahead and put the following code above void setup()

snake.ino
enum Direction {    up,    down,    left,    right};
struct Pos {    int x;    int y;};
struct Snake {    Pos positions[256]; // Array of 256 positions, representing the maximum length of the Snake's tail    int len;    Direction dir;};
struct Food {    Pos pos;};

This defines the blueprints for data packages representing position (Pos), the Snake, and the food. Position has an x and y piece representing column and row in the 16x16 matrix.

Enum is a bit less commonly seen, but it is just a way of assigning a name to a particular state that we can reference. You'll get a better idea of how these objects work as we continue.

Initializing objects#

Now that we've defined how our data is packaged, we can 'initialize' objects which hold data according to the structure's blueprint like so:

snake.ino
... // ABOVE CODEstruct Food {    Pos pos;};... // ABOVE CODE
Snake snake = {0};Food food = {0};

These objects now live in the computer's memory and will store values that we give to them according to their struct blueprint.

Finally, we just need to set the data for these objects to appropriate values at the start of our game. We can do so using this code:

snake.ino
void initGame() {    // Setup Snake    snake.len = 1;    Pos p; // Create temporary position object    p.x = 8;    p.y = 8;    snake.positions[0] = p; // Copy data from temporary position object to the snake's position object    snake.dir = Direction::left;
    // Init food    food.pos.x = rand() % WIDTH; // Sets the food to a random x position between 0-WIDTH    food.pos.y = rand() % HEIGHT; // Sets the food to a random y position between 0-HEIGHT}
void setup() {    FastLED.setBrightness(20);    FastLED.addLeds<WS2812B, DATA_PIN, GRB>(leds, NUM_LEDS);
    initGame();}
